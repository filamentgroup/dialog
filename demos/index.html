Dialog Examples

Default modal in-page dialog

A standard in-page dialog consists of an element with a class of dialog and a unique ID. The element may contain any HTML you'd like, including an optional close link, which should be denoted by the attribute data-dialog-close. Any links in the page can open that dialog by referencing its ID via their href attribute. To redirect the browser's focus back to the element that opened a dialog after it is closed, you can give the opening link a unique ID and reference it via the close link's href attribute.

When opened, the browser's URL bar will receive a hash value equal to the dialog's ID, and an overlay screen will cover all page elements just underneath the dialog. That overlay screen will close the dialog when clicked. Also, clicking the escape key or back button will close the dialog as well.

In-page dialog with transparent background

To specify that the overlay screen is transparent, but still clickable to dismiss the dialog, you can add the attribute data-transbg to the dialog div.

Ajax dialogs

A link to an external HTML file can be converted to include the external file inside a dialog instead of leaving the page. To do this, simply add the data-dialog-link attribute to any link. Note: this will pull the entire referenced document in and insert its HTML into a dialog element, and typical cross-domain Ajax restrictions will apply.

Iframed dialogs

To pull an external page into a dialog inside an iframe, you can apply the data-dialog-link and data-dialog-iframe attributes.
